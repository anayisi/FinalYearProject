UNIVERSITY OF ENERGY AND NATURAL RESOURCES, ONLINE EXAMINATION MANAGEMENT SYSTEM

About

This project presents the development of an Online Examination Management System (OEMS) tailored for the University of Energy and Natural Resources (UENR).
The OEMS aims to automate the examination process, enhance academic efficiency, and address security vulnerabilities in traditional examination environments.
The system is designed to support administrators, lecturers, and students, offering functionalities such as user management, question bank management,
exam scheduling, and result processing.

Leveraging modern web technologies, the backend is developed using PHP with the Code Igniter framework and MongoDB for database management, while the frontend
utilizes Bootstrap for responsive design. The system ensures robust security through SHA-256 encryption for user passwords. Through comprehensive testing
and evaluation, the OEMS demonstrates significant improvements in efficiency, security, and accessibility compared to traditional examination processes.
This report details the system's architecture, implementation, and potential impact on academic processes at UENR.
